# Arrow function
This is one of the most exciting features of ES6. =>It's not just a shorthand for the keyword function, it brings other 
benefits. Arrow function shares the same this with the surrounding code, which can help you to solve the problem of
 this pointing. Experienced JavaScript developers are familiar with patterns such as var self = this; or
 var that = this that refer to peripheral this. But with = > this mode is not needed.
